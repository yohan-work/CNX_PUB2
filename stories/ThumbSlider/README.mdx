# ThumbSlider Component Guide

하나의 swiper 작성만으로 썸네일 swiper를 생성해주는 component입니다.

## 설치 방법

```bash
npm install swiper
```

## 사용 방법

### HTML 기본 구조

```html
<div class="thumbswiper">
  <swiper-container class="thumbswiper_mainswiper">
    <swiper-slide>
      <img src="이미지경로" alt="이미지설명">
    </swiper-slide>
    <!-- 추가 슬라이드 -->
  </swiper-container>
</div>
```

### JavaScript 초기화

```javascript
import { ThumbSlider } from './thumb-slider.js';

const sliderElement = document.querySelector('.thumbswiper');
new ThumbSlider(sliderElement);
```

## 옵션

### swiper-container에 적용되는 옵션

<table>
  <tr>
    <th>옵션</th>
    <th>값</th>

    <th>설명</th>

  </tr>
</table>

## 예시

### 기본 사용법
```html
<div class="thumbswiper">
  <swiper-container class="thumbswiper_mainswiper">
    <swiper-slide><img src="image1.jpg" alt="Image 1"></swiper-slide>
    <swiper-slide><img src="image2.jpg" alt="Image 2"></swiper-slide>
    <swiper-slide><img src="image3.jpg" alt="Image 3"></swiper-slide>
  </swiper-container>
</div>
```

### 커스텀 옵션 사용
```html
<div class="thumbswiper" 
     thumb-slide-show="6" 
     thumb-slide-position="left" 
     thumb-swiper=".custom-thumb-swiper">
  <swiper-container class="thumbswiper_mainswiper" thumbs-swiper=".custom-thumb-swiper">
    <!-- 슬라이드 내용 -->
  </swiper-container>
</div>
```

## 주의사항

1. Swiper.js가 필요하므로 반드시 설치해야 합니다.
2. 썸네일 슬라이더는 메인 슬라이더의 내용을 자동으로 복사하여 생성됩니다.
3. `thumb-slide-position`이 `left` 또는 `right`인 경우, 썸네일 슬라이더의 높이가 메인 슬라이더의 높이에 맞춰 자동으로 조정됩니다.
